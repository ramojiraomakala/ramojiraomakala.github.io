# Ramoji Rao Makala — Personal Portfolio

A single-page personal portfolio site (static HTML/CSS/JS, no build step).

**Live:** https://ramojiraomakala.github.io

## Local preview
Open `index.html` directly in a browser, or run a local server:

```bash
python3 serve.py   # then open http://127.0.0.1:4321
```

## Deploy (GitHub Pages)
Pushed to the `ramojiraomakala.github.io` repo on the `main` branch; GitHub Pages
serves the root automatically. See repo Settings → Pages.

## Files
- `index.html` — the whole site
- `me.jpg` — profile photo
- `Ramoji_Rao_Makala_Resume.pdf` — linked from the hero "Download Résumé" button
